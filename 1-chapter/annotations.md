# *Chapter 2*

## In this chapter

* Create and use basic list functions

<p>Learn how to define and use lists operations like</p>

```haskell
-- Defining list 
myList = [0,1,2,3]

-- Basic list functions
reverse myList -- [3,2,1,0]
head myList -- [0]
last myList -- [3]
tail myList -- [1,2,3]


```
